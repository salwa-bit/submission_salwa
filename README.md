# 🎓 Digital Certificate Issuer - Soroban Smart Contract

This project is a decentralized application built on the Stellar network using Soroban. It serves as a secure, immutable ledger for issuing and verifying educational certificates.

## 🌟 Features
- **Issue Certificate:** Officially log a new certificate with the recipient's name and course details onto the blockchain.
- **Verify/View Certificates:** Retrieve the public ledger of all issued certificates.
- **Revoke Certificate:** Invalidate and remove a certificate entry if issued by mistake.
